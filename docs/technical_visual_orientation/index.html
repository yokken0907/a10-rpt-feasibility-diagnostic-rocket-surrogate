A10-RPT Reduced Rocket Surrogate
Technical Visual Orientation

This page is a repository-facing technical orientation layer for the A10-RPT reduced rocket-propulsion surrogate. It explains the mission-variable-preserving diagnostic logic, reduced nondimensional surrogate status, thermal/supply-margin/growth-rate stress structure, evidence hierarchy, and claim boundary. It is not an engine-design guide, not a flight-hardware claim, and not a propulsion safety document.

0. Reader level and purpose

This page assumes the reader has technical interest in reduced modeling, propulsion-adjacent feasibility diagnostics, control/optimization, or reproducible engineering-style archives.

The central point is not "a new rocket engine." The repository presents a nondimensional reduced surrogate for mission-variable-preserving structured-prior diagnostics under thermal, supply-margin, disturbance, and combined-stress pathways.

Correct use:
Use this page as a visual map before reading the manuscript, claim-boundary documents, figures, and reproducibility materials.

Conceptual orientation only. No propulsion simulation is executed in this HTML.

2. Reduced nondimensional surrogate

The repository should be read as a reduced-surrogate feasibility-diagnostic archive. It is not an engine geometry, injector design, combustion solver, flight code, or hardware package.

Reduced-model reading

  • The surrogate organizes diagnostic channels, not full propulsion physics.
  • Nondimensional variables make stress interactions inspectable.
  • The value is in structured comparison and claim-bounded feasibility screening.
  • Experimental engine validation and safety analysis are outside scope.
Important: Do not translate the visual diagram into construction, operation, or safety guidance.

7. Claim boundary

Allowed reading

  • Reduced rocket-propulsion surrogate study.
  • Mission-variable-preserving structured-prior diagnostic.
  • Thermal / supply-margin / growth-rate stress visualization.
  • Paper companion archive for a pre-engineering concept.

Forbidden reading

  • Flight hardware claim.
  • Engine design or construction instructions.
  • Propulsion performance guarantee.
  • Combustion solver validation.
  • Safety-certified controller.
  • Experimental rocket validation.

The purpose of this page is professional orientation: enough structure to reduce reader friction, without expanding the scientific or engineering claims.
