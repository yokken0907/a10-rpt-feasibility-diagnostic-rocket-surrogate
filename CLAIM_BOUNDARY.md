# Claim Boundary

## Allowed claims

- This repository accompanies an AI-assisted independent research manuscript on A10-RPT.
- A10-RPT is a nondimensional reduced rocket-propulsion surrogate-control framework.
- The work studies feasibility-first structured priors under thermal, supply-margin, growth-rate, noise/disturbance, and combined stress.
- The reported results support a feasibility-diagnostic interpretation within the tested surrogate.
- Resource-frontier sweeps separate controller-feasible, resource-frontier-feasible, near-frontier, and unresolved regimes.

## Forbidden claims

- This is an engine-design tool.
- This predicts real rocket-engine performance.
- This prescribes flight hardware.
- This gives instructions for hazardous experiments.
- This gives propellant-handling or engine-operation guidance.
- This is a hardware-ready controller.
- This has aerospace safety certification.
- This is a formal control-barrier proof.
- This has been calibrated against a real engine.

## Standard short wording

> A10-RPT is a nondimensional reduced-surrogate feasibility-diagnostic control study. It is not an engine-design tool, propulsion-performance predictor, hazardous-experiment guide, or hardware-ready flight controller.

## Japanese wording
